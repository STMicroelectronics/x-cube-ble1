Release Notes for X-CUBE-BLE1 Projects

Copyright © 2025 STMicroelectronics

Purpose

The X-CUBE-BLE1 Projects provide examples and applications to aid understanding how to develop an application using the BLE Middleware for the BlueNRG-M0 device. Projects include STM32CubeMX project file (.ioc) for the graphical visualization of the STM32 microcontroller pins, peripherals and middlewares configuration.
